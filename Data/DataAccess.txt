All data are available on ZENODO at https://doi.org/10.5281/zenodo.20142764. Download the "Data.zip" file and extract the contents into you main directory. 

1. ComplementaryFiles - Contents
This folder contains a vector boundary file and area extent file required to maintain consistency across the analysis and to generate the final figures.
Study Area Boundary: A shapefile (narok_county_utm.shp) defining the geographic limits of the analysis, used for clipping and as a map overlay in figures.
Extent Raster: A master template raster (extentRaster.tif) used to define the coordinate reference system (CRS), resolution, and grid alignment for all other spatial datasets to ensure pixel-level alignment.


2. Preprocessed - Contents
This directory contains the standardized environmental and land cover datasets used as inputs for the transition modeling and driver analysis. All rasters in this folder have been clipped to the study area extent and resampled to a 90m resolution.
LULC Baselines: Historical land use/land cover maps (e.g., lu2010_4cls_90m.tif and lu2018_4cls_90m.tif) reclassified into primary categories (Forest, Rangeland, Cropland, and Urban).
LULC Change factors including topographic Drivers e.g., elevation, proximity metrics e.g., euclidean distance rasters calculated for various land cover classes (e.g., dist2_cropland2000.tif, dist2_forest2000.tif), and ecological & policy factors e.g., processed layers for Forest Fragmentation, Annual Average Temperature, and Land Tenure/Policy boundaries.


3. LCM_Outputs - Contents
This folder archives the primary outputs generated by the Land Change Modeler (LCM). These files represent the raw results of the change analysis and the predictive modeling phase.
Hard Predictions: Categorical maps representing the most likely land cover state for a future date (e.g., 2018).
Soft Prediction: Vulnerability map indicating the continuous likelihood of change (e.g., _soft.rst files), used to assess model confidence.
Transition Potentials: Maps showing the modeled potential of LCLC transition for each modeled transition.
